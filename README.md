# 🔐 Password Strength Analyzer

A simple and interactive web application that analyzes password strength based on multiple security requirements.

## 🚀 Live Demo

[View Live Website](https://kumaisha-cyber.github.io/password-strength-analyzer/)

## ✨ Features

- 🔒 Real-time password strength analysis
- 📊 Score system from 0/5 to 5/5
- 🔴 Weak, 🟠 Medium, and 🟢 Strong strength levels
- 📈 Animated progress bar
- 👁️ Show / Hide password
- 🗑️ Clear password button
- 🌙 Dark mode
- 💾 Dark mode preference saved using localStorage
- 💡 Smart tips for missing password requirements
- 📱 Responsive design for mobile devices

## 🛡️ Password Requirements

The password is checked for:

- At least 8 characters
- Uppercase letter
- Lowercase letter
- Number
- Special character
